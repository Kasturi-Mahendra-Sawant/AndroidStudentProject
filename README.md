# Student Course Marks Management System (Android)

## 📌 Project Title
Student Course Marks Management System

## 📖 Project Description
The Student Course Marks Management System is an Android application developed to manage student details, course information, and marks efficiently.  
This system helps teachers or administrators to store student records, assign courses, and enter marks in a simple and organized way.

The application provides an easy interface to add students, manage courses, enter marks, and view results. It reduces manual paperwork and improves accuracy in maintaining academic records.

---

## 🎯 Objectives
- To manage student information digitally
- To store course details
- To record student marks
- To display student results easily
- To reduce manual record-keeping errors

---

## ⚙️ How to Run the Project

1. Install Android Studio on your computer.
2. Open Android Studio.
3. Click on **Open Project**.
4. Select the **StudentCourseMarks** folder.
5. Wait for Gradle Sync to finish.
6. Connect an Android device or start an emulator.
7. Click **Run** button to launch the application.
